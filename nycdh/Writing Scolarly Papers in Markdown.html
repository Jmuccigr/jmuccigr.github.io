Review

What have we got?

  • Headings
  • Paragraphs
  • Emphasis (2 flavors)
  • Lists

What more do we need?

  • Block Quotation
  • Links
  • Images
  • Footnotes

New Stuff

Block Quotation

  • Block quotations are created simply by the right angle bracket (which looks like you’re indenting the text: >
    • > This is a block quote.

  • This is a block quote.

Links

  • Links can be added using square brackets followed by round ones
    • This is a [link](http://nytimes.com/).
    • This is a link.

Images

Images can be added from the internet, or from your computer, by adding a ! in front of the link format:

  • ![Caption](link to file)
    • ![This is from the internet](https://upload.wikimedia.org/wikipedia/commons/1/1a/Cat_crying_%28Lolcat%29.jpg)
    • ![This is from my computer](/Users/john/Pictures/house.tif)

Footnotes

  • Footnotes use the square brackets (again) combined with the caret as a superscripting mnemonic.

Type 1: Like you’re used to

  • Here is some text.[^1] And then some more text after it.

    [^1]: This is the note that goes with that text. You can place it anywhere in your document (like at the bottom), as long as it’s after the paragraph.

Type 2: In-line notes

  • Here is an inline note.^[Inlines notes are easier to write, since you don’t have to pick an identifier and move down to type the note.] You might find the interruption of the text you’re writing to be distracting.
